Top-level server.js hosts Express API (/healthz, /v1/models, /v1/chat/completions) and streaming logic. auth/server.mjs implements Traefik ForwardAuth. docker-compose.yml + Dockerfile define deployment; systemd/ holds user units; config/ contains example provider configs; scripts/ house smoke + live test helpers; docs/runbooks has runbooks; tests under tests/ with Vitest + Playwright harnesses.
